# Formula 1

A project for analyzing and working with Formula 1 race data using Python.

## Data Source

Data used in this project comes from the [Formula 1 Race Data](https://www.kaggle.com/datasets/jtrotman/formula-1-race-data) dataset on Kaggle, created by jtrotman. All credit for the raw data goes to the original dataset author.
